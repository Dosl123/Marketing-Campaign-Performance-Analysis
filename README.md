# Marketing Campaign Performance Analysis (Excel)
Marketing campaign analysis using Excel (SUMIF, SUMIFS) to evaluate channel spend, location performance, and engagement metrics. Provides insights on top-performing channels, content types, and regions to support data-driven marketing decisions.

### Overview
This project analyzes a marketing campaign dataset to demonstrate how Excel functions can support real business decision-making. The focus is on evaluating spend, engagement, and performance across different channels and locations.

---

### Tools Used
- **Excel**: Data analysis using SUMIF and SUMIFS functions  

---

### Analysis Performed
Using Excel functions, the following areas were analyzed:
- Spend across social media platforms  
- Performance by location  
- Engagement metrics such as clicks and impressions  

---

### Reports Created
- Marketing Summary  
- Channel Spend Performance  
- Channel Clicks Performance  

---

### Key Insights
- YouTube generated the highest impressions and clicks, making it the strongest channel for visibility  
- Phoenix recorded the highest clicks, followed by Los Angeles  
- Text-based content outperformed video and image formats in both clicks and cost efficiency  
- Houston showed relatively lower engagement compared to other locations  

---

### Conclusion
Excel functions such as SUMIF and SUMIFS enable quick and effective breakdown of data across multiple dimensions, including channel, location, and content type. This supports clear insight generation and better marketing decision-making.
